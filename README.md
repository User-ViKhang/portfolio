## About me
I am a Junior Quality Automation Engineer with hands-on experience in both manual and automation testing. 
I started my QA journey in July 2024, working on retail desktop applications including BOM, Central, and POS systems. 

I have experience designing and executing test cases, performing API testing, and developing automation scripts using Playwright and Robot Framework. 
I have also worked on web, mobile, and Web3 projects, integrating automated test workflows with tools like GitHub Actions. 

With a strong attention to detail and a logical testing mindset, I am passionate about ensuring product quality and continuously improving my automation skills to grow as a QA Engineer.
